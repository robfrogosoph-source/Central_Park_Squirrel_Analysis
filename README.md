## How to use
- To browse locally, open `README.md` or any file in your browser from the project root.
- Open the chart files directly, for example: `visuals/central_park_map.html`.
- If you want a simple server, run `python -m http.server` from the project root and open `http://127.0.0.1:8000`.
- For GitHub Pages, publish this repository and make sure the `visuals/` folder is included in your branch.
